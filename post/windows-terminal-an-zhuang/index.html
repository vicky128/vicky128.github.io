Windows Terminal安装

2022-01-12 3 min read

昨天弄Ubuntu的时候再次看到有关Windows Terminal美化的视频，上一次是在少数派，我也是真的不喜欢默认的CMD或者windows power shell页面，真的太丑，但是那个时候下载Windows Terminal失败了，今天再次遇见，那我肯定要搞啊。

这次时间充裕，正儿八经按照步骤来排查原因（通过GitHub下载了Windows Terminal Preview和Windows Terminal，安装后启动不了，怎么也打不开，就报错说响应时间太长了），其实我的电脑版本都是符合的，然后偶然看到有一个视频评论说微软商店下载不了是因为Windows自动更新禁止了，我一想还真是，因为之前Windows自动更新win10以后我的IDEA出错了，拿去修排查出来可能是因为系统的问题，然后回退回之前的系统才成功，然后维修人员跟我说一般还是等稳定一段时间以后再更新，频繁要用电脑的话尽量不更新，然后就把系统的自动更新设置成时间最长的提醒，每隔一段时间再去设置一次。

因为想说今天真的要尽全力把它搞出来，所以就想更新系统试试吧，也不更成win11，就win10。没想到到Windows更新那儿特别卡根本改不了之前的设置。然后按照网上的疑难解答排查，最后终于不卡了，但是检查更新报错Error 0x80080005，中文谷歌没有答案，百度也没有，换成英语谷歌才在油管看到解决方法。

最后终于更新成功，然后Windows Terminal也成功打开了。最后也按照美化的视频弄出我满意的界面了。小插曲：Oh My Posh里面的主题下出来与预览不同，说是要改字体，但是在Nerd Fonts里面下载的字体还是没有解决问题，又去找说是要满足powerline，中间兜兜转转，发现字体要选择含nerd font的才行。

其实中途折腾的时候搞得Ubuntu原本是找到路径的，但是突然\wsl$不起作用了，后来Windows Terminal成功以后又能打开了。不知道IDEA会不会又出问题🤔

本来只是在Linux里面解压文件失败了，想说图形界面操作吧，搞了这么一出，解压只有到明天了😳
